# NeuroAbility Foundation

  Website design mockup for the NeuroAbility Foundation — a nonprofit focused on accessibility research, open-source tools, and connecting students with neurological/chronic health conditions to free 1:1 mentors.

  ## Pages
  - **Landing** — mission, action cards (Donate, Support a Student, Learn How to Help)
  - **Donate** — monthly/one-time giving with donation tiers
  - **Refer a Student** — warm referral form for educators, parents, counselors
  - **Guides** — resource library for educators, parents, and mentors
  - **Find a Mentor** — mentor matching request form + mentor spotlight

  ## Stack
  Built as interactive React mockups in a pnpm monorepo (Vite + Tailwind + Framer Motion).
